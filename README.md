# Контейнеризация на Web App с Docker Compose

Този проект представлява просто Node.js уеб приложение, свързано с MongoDB база данни, контейнеризирано с Docker.

## Структура на проекта
- `server.js` - Основен бекенд код на приложението (Express сървър).
- `Dockerfile` - Конфигурация за изграждане на Docker образа на уеб сървъра.
- `compose.yml` - Конфигурация за стартиране на уеб сървъра и MongoDB в обща мрежа.

## Компоненти и Комуникация
1. **web-app**: Node.js приложение, което слуша на порт 3000.
2. **database**: Официален Docker образ на MongoDB, работещ на порт 27017.
Комуникацията се осъществява чрез вътрешната Docker мрежа, като бекендът се свързва към базата чрез хост името `database`.
